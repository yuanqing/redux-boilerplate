# redux-boilerplate

## Usage

Clone and run [this gist](https://gist.github.com/yuanqing/bba830e9d05c8875b45f). Then:

```
$ git clone https://github.com/yuanqing/redux-boilerplate.git
$ cd redux-boilerplate
$ npm install
$ npm start
```
